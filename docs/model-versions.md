# Model versions

`MULTIFACTOR_V0` version 1.0.0 targets a 1–3 month horizon: quality 25%, value 20%, growth 20%, momentum 35%. All required factor scores must exist; confidence never enters alpha and risk/liquidity remain separate. Candidate is top 10%, hold range top 20%, neutral 70–80th percentile, and exit range below the 70th percentile. These are model classifications, not trade instructions.

`MULTIFACTOR_V1` is configuration-only and inactive: quality 20%, value 15%, growth 15%, revisions 25%, momentum 25%. It cannot activate until dependable point-in-time revisions exist.
