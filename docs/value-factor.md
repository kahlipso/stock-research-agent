# Value factor

Value has 50% minimum weighted coverage. It uses positive earnings yield, FCF yield, EBIT/EV, EBITDA/EV, sales/EV, applicable book-to-market, shareholder yield and historical valuation percentile. EV ratios require positive enterprise value; negative earnings are not classified as cheap. Operating EV ratios are excluded for financial industries. Forward estimates, PEG, targets and fabricated consensus inputs are prohibited.
