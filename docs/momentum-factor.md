# Momentum factor

Momentum has 75% minimum weighted coverage. Features are one- and three-month return, six-minus-one and twelve-minus-one return, relative returns against stored SPY/QQQ/sector bars where present, volatility-adjusted momentum, distance from the 200-observation mean and 252-observation high, short-term reversal, and volume trend. The implementation uses adjusted close, exact trading observations, rejects insufficient history, does not interpolate and stores source endpoints. One-month return has 2% weight.
